Trabalhando com listas

Listas ordenadas

Desse jeito, você criará listas numeradas, exemplo:

  1. Acordar
  2. Ligar pro João
  3. Tomar café

Tem como criar listas por letras também, colocando um type no <ol>:

  1. Acordar
  2. Ligar pro João
  3. Tomar café

Esse type tem umas 5 opções, só escolher uma.

Listas não ordenadas

São listas sem sequência, usando <ul>, exemplo:

  • Arroz
  • Feijão
  • Biscoito

E da pra mudar o tipo também se não quiser uma bolinha ali.
E claro, da pra por uma lista dentro da outra.

Lista de definição

Serve para estruturar pares de termos e suas respectivas descrições, como glossários, catálogos de produtos, glossários, Perguntas Frequentes (FAQ) ou metadados de chave-valor.
Ela utiliza as tags <dl> (lista), <dt> (termo) e <dd> (descrição).

Como funcionam os links - externos

Link que não abre em outra aba

É recomendado que você crie links que abram na sua mesma aba apenas quando é a continuação de sua site seu, ou algo seu.
Se não for, crie links com <a target="_black"> para o usuário ser redimencionado para outra aba.

Link internos

Um link interno te redireciona para lugares dentro do seu próprio site.

Tipos de degradê em CSS

Existem vários tipos, por exemplo:

TO BOTTOM

background-image: linear-gradient(to bottom, white, rgb(227, 178, 246));
Ele vai fazer o degradê de cima pra baixo.

TO TOP

background-image: linear-gradient(to top, white, rgb(227, 178, 246));
Ele vai fazer o degradê de baixo pra cima.

TO LEFT

background-image: linear-gradient(to left, white, rgb(227, 178, 246));
Ele vai fazer o degradê da esquerda pra direita.

TO RIGHT

background-image: linear-gradient(to right, white, rgb(227, 178, 246));
Ele vai fazer o degradê da direita pra esquerda.

É possível colocar quantas cores quiser, não apenas 2.
